distâncias absolutas. < Length >

CM
Centímeto. 1cm = 96px/2.54
IN
Inches (polegadas). 1in = 2.54cm = 96px
PX
Pixels. 1px = 1/96th of 1in

Distâncias relativas

São relativas a algum outro valor, pode ser o elemento pai, ou root, ou o tamanho da tela.

EM
Tamanho da font do pai. (Atenção, ele multiplica pelo valor adicionado, por exemplo: 1.3em seria vezes 1.3.
REM
Tamanho da font do elemento raiz (root/html).
Por padrão o elemento root ou html vem com a font size em 16px.
Também podemos mudar o elemento HTML ou ROOT, seria:

:root{
}

ou

html{
}
VW, (viewport é toda a tela branca da navegação)
1% da viewport width(largura).
VH
1% da viewport height(altura).
