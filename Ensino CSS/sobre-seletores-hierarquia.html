O que é um seletor?

Os seletores são o que conectam um elemento HTML com o CSS, existem vários tipos, inclusive, nós vimos um na última aula, o Element/Type selector, mas também temos o seletor global, que é um * (asterisco), ID selector, que é # (cerquilha, cardinal) e o ID do elemento HTML, class selector, que é um . (ponto) e o nome da classe, e mais alguns que podemos entender mais tarde no curso. Os seletores que mais usaremos serão realmente apenas os anteriormente citados, e vamos entrar em exemplos de como usá-los: HTML

Atenção

Quando usamos ID e CLASS no elemento e ele estiver com css ele recebe a edição dos dois.

Exemplo abaixo:
CSS /* ID selector */ #container { width: 200px; } /* Class selector */ .m-40 { margin: 40px; } /* Element/Type selector + Agrupamento de seletores */ h1, h2 { color: blue; font-size: 60px; background: gray; }
