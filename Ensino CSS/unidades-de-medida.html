Tipos numéricos.

Integer
Número inteiro como -10 ou 223
Number
Número decimal como -2.4, 64 ou 0.234
Dimension
É um < number > com uma unidade junto: 90deg(graus), 2s, 8px
Percentagem
Representa a fração de outro número: 50%

Unidades comuns

length
Representa um valor de distância: px, em, vw
Angle
Representa um ângulo: deg(Graus), rad(radianos), turn(Giros_
Resolution
Representa resoluções para dispositivos: DPI.
