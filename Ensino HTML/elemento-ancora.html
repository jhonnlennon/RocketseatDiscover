HyperLinks - Elemento Âncora

Atributos

Globais
São os atributos globais: Title, ID, Class, dentre outros...

href
Para onde iremos quando clicarmos no link
- url completa: http://www.google.com.br
- fragmento: Tudo vindo do #, exemplo: #sobremim, agora, caso tenha alguma tag com o ID com "sobremim" a tela vai ir até ela.
- email: mailto:jhonn1990@hotmail.com
- telefone: tel:+5524981620500
- Outros...

Download
Podemos usar o atributo "Download", assim faz o download da página ou imagem... o que for..

Target
- _self (padrão), ao clicar, troca a página que esta pela a que clicou..
- _blank, abre uma nova aba no navegador.
