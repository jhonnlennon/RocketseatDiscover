Sobre tabelas

Informações:

Tag TABLE
Usado para dizer que se inicia uma tabela.

Tag CAPTION
Usando para adicionar uma descrição da tabela.

Tag THEAD
Usando para colocar as linhas e colunas dentro do cabeçalho.

Tag TBODY
Usada para indicar as linhas e colunas dentro do corpo da tabela.

Tag TFOOT
Usanda para indicar as linhas e colunas dentro do rodapé da tabela.

Tag TR
Significa Table Row, ou seja LINHA.

Tag TH
Significa Table Head, ou seja CABEÇALHO.

Tag TD
Significa Table Data, ou seja INFORMAÇÃO, ela cria uma célula na tabela.
