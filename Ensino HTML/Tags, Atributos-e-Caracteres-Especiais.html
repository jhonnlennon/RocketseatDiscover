#Tags#

Emmet Abreviation: É um recurso do VSCODE, utilizado como um atalho para a programação.

Digitando uma referencia de uma tag, por exemplo, p ou h1, aparecerá uma opção de autocomplemento. É mais rápido.

Ao logo do curso, as TAGS serão apresentadas, conforme seu devido aparecimento, implento sua tag e função.

A tag em sí é um elemento e pode ser elemento oculto. Por exemplo a tag IMG, que tem seus atributos porém quando a utilizamos não aparece nada em tela até que seja preenchido seu atributos quando o caso.

  • Tags: H1 até H6
      • Altera o tamanho das fontes
  • Tag: ol e ul
      • OL: Uma lista ordenada
      UL: Uma lista desordenada


    #Atributos#

    Os atributos servem como informações extras, por exemplo quando faltar a imagem aparecer um nome. (Boas práticas)

    Ou como configurações.

    É importante salvar quanto aos atributos booleanoes, eles não precisam de conteúdo. Exemplo: "disable"


    Vamos agora falar sobre os atributos Globais.

  • class
      • Além de usarmos no JS, também podemos acrescentar CSS.
  • contenteditable
      • Atributo que permite alterar o conteúdo da tag, Exemplo:

    Este conteúdo aqui vai poder ser editado!

  • data-*
      • Utilizado para tratamento no JS. Em breve mais.
  • hidden
      • Esconde uma TAG completamente.
  • id
      • Utilizado para identificação.
  • style
      • Utilizado para aplicar estilo na tag. Mas não se usa no HTML. Usa as classes no CSS
  • tabindex
      • Muito legal, quando estivermos navegando usando TAB, por exemplo, página de login, ele vai determinar as ordens do tab.
